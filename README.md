## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of HTML/CSS (for modifications)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Urpirio/Portafolio.git
```

2. Navigate to the project directory:
```bash
cd Portafolio
```

3. Open the main page in your browser:
```bash
# Open Zpages/index.html in your preferred browser
# On Linux/Mac:
open Zpages/index.html
# On Windows:
start Zpages/index.html
```

### Development

To make modifications:
1. Edit HTML files in the `Zpages/` directory
2. Modify styles in the `YStyle/` directory
3. Add or update images in the `URecursos*/` directories
4. Test changes locally by opening HTML files in your browser
